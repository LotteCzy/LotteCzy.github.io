maven

1. 用来干嘛

Maven 翻译为”专家”、”内行”，是 Apache 下的一个纯 Java 开发的开源项目。基于项目对象模型（缩写：POM）概念，Maven利用一个中央信息片断能管理一个项目的构建、报告和文档等步骤。

Maven 是一个项目管理工具，可以对 Java 项目进行构建、依赖管理。

Maven 也可被用于构建和管理各种项目，例如 C#，Ruby，Scala 和其他语言编写的项目。Maven 曾是 Jakarta 项目的子项目，现为由 Apache 软件基金会主持的独立 Apache 项目。

3. idea的maven配置


如图打开idea中File>Setting>Build,Execution DEployment>Build Tools>Maven

  1. 将Maven home directory修改为你安装maven路径
  2. 将User setting file 修改为你刚才修改的配置文件（2）
  3. 将local repository 修改为你刚才配置仓库路径

接下来就要新建maven项目了，点击File>new>project

点击next并修改项目名称Name、项目路径Location、项目组名GroupId(我还不知道是什么)、项目模块名称Artifictid、版本号Version
